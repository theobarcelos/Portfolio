CoinZ

Site realizado para ser apresentado como Projeto do Ano Letivo, matéria específica para a realização de um trabalho ao longo do ano, no Senac Sorocaba. Nele, foi utilizado HTML5 e CSS3, apenas. Inteiramente realizado por mim. O CoinZ tem a premissa de trazer conhecimento da área financeira para todos os jovens que são leigos no assunto. No site são mostrados vídeos e notícias para a adição de conhecimento financeiro, juntamente com uma demonstração de página em manutenção, que seria a página sobre Imposto de Renda. O site foi inteiramente pensado como mobile-first, porém a versão para computador ainda não foi pensada.

Mi-Au

Site realizado para ser apresentado também como Projeto do Ano Letivo do Senac Sorocaba. O diferencial é que este projeto não é meu, e sim de amigas da minha sala, com a premissa de resgatar e acolher animais abandonados por Sorocaba e região. A parte de HTML5 e JavaScript foi realizada pelas garotas do grupo, já o CSS3 foi realizado por mim, principalmente a seção de “Contato”. O projeto foi inteiramente pensado para computadores apenas, sem previsões para a realização de uma versão para celular.

Neurahub

Este código foi realizado por mim, para o projeto de um grande amigo, Marco Túlio. Ele utiliza Python, mais precisamente, a biblioteca do BeautifulSoup. Este código é nada mais, nada menos que a automação de entrega de informação resumida para o aluno que quiser saber de qualquer assunto. O código raspa os dados diretamente da Wikipédia, e entrega com uma quantidade certa de caracteres, para que o aluno não receba toda a informação que a página passar, que seria coisa até demais.
