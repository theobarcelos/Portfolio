CoinZ

Neurahub

Este código foi realizado por mim, para o projeto de um grande amigo, Marco Túlio. Ele utiliza Python, mais precisamente, a biblioteca do BeautifulSoup. Este código é nada mais, nada menos que a automação de entrega de informação resumida para o aluno que quiser saber de qualquer assunto. O código raspa os dados diretamente da Wikipédia, e entrega com uma quantidade certa de caracteres, para que o aluno não receba toda a informação que a página passar, que seria coisa até demais.
